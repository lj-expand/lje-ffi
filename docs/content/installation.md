# Installation

1. Download the latest release from the [GitHub releases page](https://github.com/lj-expand/lje-ffi/releases/latest)
2. Make sure to download BOTH `libffi-8.dll` (once) and `lje-ffi.dll`
3. Put both in `%USERPROFILE%\.lje\binaries`
4. You only need to update `lje-ffi.dll` when a new version comes out, `libffi-8.dll` should work for all versions of LJE-FFI.
